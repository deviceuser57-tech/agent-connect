# Lovable Smoke Test Checklist

After pushing to GitHub (Lovable auto-deploy), verify:

1. App loads without a blank screen and shows the main layout.
2. Sign-in flow works and user session persists on refresh.
3. AI Chat page loads and can send a message (expect response or friendly error).
4. Agent Test Chat page loads and can send a message (expect response or friendly error).
5. Knowledge Base loads and document upload shows progress and completion.
6. Multi-Agent Canvas loads, can open/create a workflow, and save without errors.
7. Workflow Run from canvas shows a completion message or a friendly error.
8. Marketplace and Team pages render without crashes.
